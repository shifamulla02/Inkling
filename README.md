# INKLING

A fast, secure, and easy-to-use online PDF tool suite for managing PDF documents directly in your browser.

PDF Toolkit provides essential PDF utilities such as merging, splitting, compressing, converting, and securing PDF files without requiring software installation.

## Features

### PDF Editing & Organization

- Merge multiple PDF files into one document
- Split PDFs by page range
- Extract selected pages
- Reorder pages using drag-and-drop
- Rotate PDF pages

### PDF Conversion

- PDF to Images
- Images to PDF

### PDF/WORD Extraction

- PDF Extraction
- WORD Extraction

### PDF Optimization

- Compress PDF files
- Text Watermark
- Image Watermark

### Security

- Protect PDFs with passwords
- Remove PDF passwords (with proper authorization)
- PDF Metadata

### User Experience

- Drag-and-drop file uploads
- Mobile-friendly interface
- Fast processing
- Clean and simple design
- No account required
